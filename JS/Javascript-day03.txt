	1. Javascript是动态类型语言， 变量的类型可以根据其值随时变化；所以变量在赋值之前是没有类型的，譬如变量的声明只有一个关键字，只声明不赋值的时候是没有类型的，其包含的方法和属性也是只有原型Object的属性。所以在多次重复声明中，在以后的声明中只要没有赋值，变量的值是不会变化的，且在声明时可以只声明其类型，这是为了使用其方法和属性，=》类型是和方法和属性绑定在一起的，变量的值也只是提供了属性的依据和方法的操作对象。
	2. Javascript 中所有的事物都是对象，内在对象都是冉ǖ睦嘈投加冉ǖ姆椒ǎ可以使用new创建自己的对象或者类型，new Object()是创建自定义对象，然后使用.方法来创建自定义对象的属性和方法，通过new 构造函数(）的方法来创建自己的类型，然后仍然可以改造此属性或者方法。
	3. javascript对象引用方法，属性和设置属性方法
	引用：
	objectName.propertyName
	objectName.propertyMethod()
	
	设置：
	objectName.propertyName=值
	objectName.propertyMethod = 函数
	4. Camel-case标记法来命名javascript名称
	5. Javascript 函数的参数定义时可不必声明，使用时可以从arguments数组中取得。
	6. 函数的返回值是可选的，当有返回值时，函数返回时将把值从函数体内传递到函数调用处。
	7. Javascript 变量的生命期从他们被声明开始，在函数体内通过关键字var声明的变量是局部变量，在函数退出后，局部变量就被删除。全局变量包括不在函数内部声明的变量和在函数内部未通过关键字var声明的变量；全局变量会在页面关闭后被删除
	==》在函数内部直接使用而未使用关键字var来声明，默认就是全局的变量，所以其就是全局的变量，没有也要造一个出来。
