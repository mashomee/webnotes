DOM Tree
* js操作domTree
  通过dom树，js可以增删改查页面中的所有html元素、可以增删改查页面中所有的html属性，
  可以增删改查页面中所有的css属性，能够对页面中所有的时间做出反应。
  （元素、属性、css Style，事件）
* 查找html元素
  - 通过元素的id属性
  - 通过元素的标签名称
  - 通过元素的类型即元素的class属性
* 更改html内容
  - 直接改变html输出流
  - 查找到特定元素改变此元素下的domTree
  - 查找到特定元素改变此元素的属性(在domTree中此元素的属性即是在js中此元素对应的对象的属性)
* 更改htmlTag的CSS样式
  - 外部链接的CSS样式表和内部内联的CSS样式以及每个元素指定css属性都会在html载入后构造成一颗css属性树
  - 通过每个tag对应的domTree对象的style属性可以查询综合在一起的css属性。
  - 如果单独通过每个tag对应的style属性来设置和查询的话就只能查到通过tag的style属性这一途径设置的属性，
    在外部样式表和内联样式表中设置的属性都会查询不到。
* 更改html事件处理过程
  - 事件属性本来就存在，如果需要处理的话只需要加上js代码即可，
  - 添加事件处理代码的时候可以在html文件中标签事件属性的值域直接处理事件代码，
    也可以在此处调用js函数，在函数内处理事件，也可以在其他的js代码内处理事件属性的增删改查
