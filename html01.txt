	1. Html is a set of markup tags, used to describe web pages.
	2. Html documents is called web pages.  Web browsers is used to display html documents as web pages.  Markup tags is used to interpret the content of the page.
	3. Html documents are defined by html elements.
	4. Html elements are a pair of start /opening tag and end/closing tag.
	5. Html element content are everything between start tag and end tag.
	6. Html element can be empty, then start tag is also end tag.
	7. Html element can have attributes. They reside in start tag, and come in name/value pairs like [name='value'].
	8. CSS styling can be added in the following three way:
		a. In a separate style sheet file, normally .css files
		b. In the "style" element in the html head element.
		c. In the style attribute in a single html element.
